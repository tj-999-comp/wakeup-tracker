# 覚醒トラッカー

## 概要
このプロジェクトは、iPhone ショートカットを使って覚醒時間を記録し、曜日や休日の選択を Google Sheets に保存し、毎週月曜日に視覚的に洗練された HTML レポートを送信するシステムです。

## ワークフロー
1. **ユーザーが覚醒時間をタップ**: iPhone ショートカットを使用して覚醒時間を記録します。
2. **曜日/休日を選択**: ユーザーは、タップ時に曜日または休日を選択します。
3. **データの保存**: 記録されたデータは Google Sheets に保存されます。
4. **レポート生成**: 毎週月曜日に、Google Sheets からデータを取得し、視覚的な HTML レポートを生成します。
5. **レポートの送信**: 生成されたレポートを指定されたメールアドレスに送信します。

## データモデル
- **タップ時間**: UTC タイムスタンプで記録
- **曜日/休日**: 選択肢として保存
- **ユーザー情報**: メールアドレスやその他の必要な詳細

## 週次レポートメトリクス
- 合計タップ数
- 各曜日のタップ分布
- 休日のタップ数
- トレンド分析（前週との比較）

## アーキテクチャ
- **フロントエンド**: iPhone ショートカット
- **バックエンド**: Google Sheets API, および HTML レポート生成
- **メール送信**: SMTP サーバー経由で送信

## ロードマップ
1. **フェーズ 1**: 基本的な覚醒時間の記録機能を実装
2. **フェーズ 2**: Google Sheets へのデータ保存機能を実装
3. **フェーズ 3**: 週次レポートの生成と送信機能を実装
4. **フェーズ 4**: UI の改善と使用感の向上
