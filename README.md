# Wakeup Tracker

## 概要
Wakeup Tracker は、iPhone のショートカットをウィジェットから起動し、起床したタイミングを手軽に記録するための個人用ツールです。

記録する内容は、次の 2 つだけです。

- 起床時刻（ショートカットを実行した時刻）
- 平日 / 休日

記録先は Google スプレッドシートを想定しています。  
毎週月曜日に、1週間分のデータを集計した HTML 形式のレポートをメールで送信します。

---

## 想定ワークフロー
1. iPhone のウィジェットからショートカットを起動する
2. 「平日」または「休日」を選択する
3. 実行時刻と選択内容を Google スプレッドシートへ記録する
4. 毎週月曜日に集計処理を実行する
5. HTML 形式の週次レポートをメールで送信する

---

## 記録するデータ
このプロジェクトでは、入力項目をできるだけシンプルに保ちます。

- `recorded_at`: 記録日時
- `wakeup_time`: 起床時刻（ショートカット実行時刻）
- `day_type`: 平日 / 休日

※ 実装上は `recorded_at` と `wakeup_time` が同じ値になる想定です。  
将来的に拡張しやすいよう、項目は分けて扱える設計にします。

---

## レポート内容
毎週月曜日に送信するレポートでは、以下を確認できるようにします。

- 平日平均起床時刻
- 休日平均起床時刻
- 最早起床日
- 最遅起床日
- 記録日数

必要に応じて、各記録の一覧もあわせて表示できるようにします。

---

## 想定構成
- **入力**: iPhone ショートカット（ウィジェット起動）
- **保存先**: Google スプレッドシート
- **集計/送信**: Google Apps Script
- **出力**: HTML メール

---

## 実装方針
### 1. 起床記録
- iPhone のショートカットをウィジェットから起動する
- 起動後に「平日 / 休日」を選択する
- 実行時刻をそのまま起床時刻として扱う

### 2. データ保存
- ショートカットから Google Apps Script の Web アプリへ送信する
- Apps Script で Google スプレッドシートに 1 行追加する

### 3. 週次レポート
- 毎週月曜日に Apps Script の時間主導トリガーで集計処理を実行する
- 直近 1 週間の記録を対象に集計する
- HTML 形式で見やすいメールを生成し、指定アドレスへ送信する
